Git For Windows
###############

..  include::   /references.inc

Git_ is available for installation on Windows systems in a simple installation package.

Downloading Git
***************

Here is the link to the download page:

    * `Git For Windows <https://github.com/git-for-windows/git/releases/latest>`_

Installing Git
**************

Once the installer is on your system, navigate to the file probably in your
``Downloads`` folder, and double-click on the file name. The program should
install with not issues.
